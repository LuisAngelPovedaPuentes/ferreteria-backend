# 🛠️ Sistema de Gestión - Ferretería La Economía A y G (Backend)

Backend desarrollado para la gestión de productos, proveedores y usuarios en una ferretería.

Este proyecto forma parte de mi proceso de formación como desarrollador backend.

---

## 🏗️ Arquitectura

El proyecto está basado en **Arquitectura Hexagonal (Puertos y Adaptadores)**.

- **Dominio**: lógica de negocio  
- **Aplicación**: casos de uso  
- **Infraestructura**: controladores y base de datos  

---

## 🚀 Tecnologías

- Java 17  
- Spring Boot  
- MySQL  
- Gradle  

---

## ⚙️ Funcionalidades

- CRUD de productos  
- CRUD de proveedores  
- API REST
