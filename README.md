# 🛠️ Sistema de Gestión - Ferretería La Economía A y G (Backend)

Backend desarrollado para la gestión de productos, proveedores y usuarios en una ferretería.

Este proyecto forma parte de mi proceso de formación como desarrollador backend.

---

## 🏗️ Arquitectura

El proyecto está basado en **Arquitectura Hexagonal (Ports & Adapters)**, lo que permite separar la lógica de negocio de las tecnologías externas.

Estructura:

- **Domain**: Entidades y lógica de negocio  
- **Application**: Casos de uso y puertos  
- **Infrastructure**: Controladores REST, repositorios y configuración  

---

## 🚀 Tecnologías

- Java 17  
- Spring Boot  
- Spring Data JPA  
- MySQL  
- Gradle  

---

## ⚙️ Funcionalidades

- CRUD de productos  
- CRUD de proveedores  
- Gestión de usuarios  
- API REST
